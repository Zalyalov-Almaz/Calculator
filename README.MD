## Приложение "Калькулятор"

### Описание
Приложение читает из консоли aрифметические операции и выводит в консоль результат их выполнения.


### Спецификация
+ Калькулятор умеет выполнять операции сложения, вычитания, умножения и деления с двумя числами: a + b, a - b, a * b, a / b.
+ Данные передаются в одну строку. Выражение не должно содержать пробелов и других посторонних символов.
+ Калькулятор умеет работать как с арабскими (1, 2, 3, 4, 5…), так и с римскими (I, II, III, IV, V…) числами.
+ Данные передаются в одну строку. Выражение не должно содержать пробелов и других посторонних символов.
